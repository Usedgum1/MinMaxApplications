Past Projects

Excel Data Trimmer

This tool was born out of the need to save a Sales team time. Their system would output reports that could have upwards of 250+ columns and can contain 20k+ lines of data. The trimmer can take any Excel file and dynamically read the headers. It will present a list of all columns within that excel sheet and allow you to checkmark which ones are important. It will then provide you with a new file of just the columns you want. It includes a smart search function and presets that you can save/load. This allows the sales team to create reports, based on large data, within seconds that would otherwise take days. Presets can also be shared with others.

Shipment Consolidator

The shipment consolidator program is a tool that will accept a file of logistic loads that an existing system outputs. It will parse the loads and based on the user’s selected choices, consolidate shipments that could/should have went out on the same day. A real example was a report of 500 loads, and it found 24 consolidation opportunities within 3 seconds. This is a task a sales team had to do by hand. Now a 6-day project can be completed in half an hour. Not only does this save time, but their customers now have more cost-saving opportunities that could’ve otherwise been missed from human error.

Carrier On Time Report Merger

This was a requested tool with a specific purpose. This report takes an existing data file and files provided by LTL carriers and cross references loads. If a load from the carrier report has a delay or service code, it generates under the appropriate line of a data file for that load and goes into a newly created column. This tool can handle several carriers, and new ones can be added if needed. End users no longer need to manually look up each code and find the existing loads to label them.

Email Splitter

This was a tool requested by a marketing team. This was a simple tool created in just a few hours. The concept is that you can upload a document with thousands and thousands of emails, and the application will separate them into easy to “copy and paste” lines based on the character limit you choose. Then a new file will be saved with lines of emails that match your character count limit. The new lines can be used to blast out emails on a grand scale. They no longer need to count out by hand or guess how many emails to include in each email chunk. The smart logic will know if a split occurs in the middle of an email and will handle it appropriately by including that spliced email on the next line of the output file. This ensures all emails are included and match the required rules without missing or breaking emails.

Carrier Pricing Parser

This tool will take a specific carrier’s pricing agreement and read through it then dump all the needed data into an easy-to-read format. It will create every variation of logistic lane provided with their appropriate pricing attached to it. This tool is used by a pricing team for building logistic lanes.

Quote Parser

This tool was built to eliminate the need to manually look up information in large excel files. These excel files had thousands of lines of quoted projects. This tool takes a user input of a dollar value and builds a report of all quotes that meet that criteria. Gone are the days of wasted hours manually extracting data from large Excel files. This turns a large project into a single step of clicking a button.
